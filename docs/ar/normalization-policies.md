# سياسات التطبيع

السياسات تعبّر عن نية الاستخدام، وليست مجرد درجات تنظيف.

## Strict

أقل تدخل ممكن. يحافظ على النص قدر الإمكان. مناسب للنصوص التي تحتاج حفظًا قريبًا جدًا من الأصل.

## Display

تنظيف خفيف للنص المعروض للمستخدم. يضبط Unicode والمسافات ويزيل التطويل دون طيّ الحروف العربية.

## Slug

لإنشاء نص صالح للرابط وقابل للقراءة. يزيل التشكيل والتطويل ويضبط المسافات والأرقام، لكنه يحافظ على الهوية الإملائية في الروابط العربية مثل `ة` و`ى` و`إ` و`ئ`.

## Search

تطبيع قوي للبحث والمطابقة. يزيل العلامات ويطبع صور الحروف ويطبع الأرقام ويحوّل الحروف اللاتينية إلى lowercase. لا تستخدم ناتجه للعرض.

## Security

يزيل محارف Unicode المخفية ومحارف التحكم في الاتجاه مع الحفاظ على الإملاء الطبيعي.

```php
use ArabicSupport\Arabic;
use ArabicSupport\Enums\ArabicPolicy;

Arabic::normalize('مُدَرِّسَةٌ عَلَى', ArabicPolicy::Display);
Arabic::normalize('مُدَرِّسَةٌ عَلَى', ArabicPolicy::Search);
Arabic::normalize('قائِمةٌ تَجْرِيبيّة', ArabicPolicy::Slug);
```
